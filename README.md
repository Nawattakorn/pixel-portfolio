# Pixel Portfolio

This repository contains a Vite + React portfolio project.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build

`npm run build`

## Deploy to GitHub Pages

This project is configured with GitHub Actions in `.github/workflows/deploy-pages.yml`.
